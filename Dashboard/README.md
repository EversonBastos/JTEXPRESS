# Atendimentos.csv

Este arquivo contém a base de dados utilizada no dashboard do projeto.

Os dados utilizados no Dashboard são simulados.

Para proteção das informações e atendimento à LGPD, os dados foram agrupados por categoria e quantidade, sem identificação de clientes, telefones, endereços, CPF/CNPJ, códigos de rastreio ou prints de conversas.

A finalidade deste arquivo é realizar uma análise de dados para identificar as dúvidas mais recorrentes dos clientes e apoiar a melhoria do atendimento da empresa. Essa análise contribui para manter a qualidade do atendimento durante a nova transição da equipe comercial, em que os vendedores internos passaram a atuar de forma externa.

Com esses dados, o projeto busca auxiliar na construção do chatbot, organizar as informações mais solicitadas pelos clientes e alimentar o dashboard de indicadores.
